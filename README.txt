Pre requisite for running the project
1)	Maven should be installed
2)	JDK

How to run test cases
mvn clean install

Where to see test report.
1)	After execution is complete, refresh log folder
2)	Open File “ResultFile07-12-2020153016.xml” in Internet explorer to view report.
3)	Accept all blocked content by clicking “Allow”
4)	Hover and click on each step to expand and go in details.
